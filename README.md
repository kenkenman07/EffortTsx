## 概要

**Efforest** は、友だちのオンライン状況を見ながら、集中時間を共有できる Web アプリです。  
**作業時間の可視化・一体感**を UI で実現しています。

- **タイマー**
- **Presence**：オンライン中のフレンドが**一目で分かる**
